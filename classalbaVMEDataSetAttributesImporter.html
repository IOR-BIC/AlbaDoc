Detailed Description

albaVMEDataSetAttributesImporter

Import generic time varying data set attributes in a vme. The input vme is modified ie all operations occurs in place. Actually supports vme mesh, surface and volumes

◆ UseIdArrayOn()

void albaVMEDataSetAttributesImporter::UseIdArrayOn ( )
inline

An id array can be used to map vtk cell or points ID to special ids, this is useful for interfacing with finite elements simulation software for example the following input attributes file:

ELEM EPEL1 49445 0.0004 41871 0.0035 41995 0.0045

has elements with ID 49445, 41871, 41995

if this array is present in vtk data structure:

ANSYS_ELEMENT_ID 1 3 int 41995 49740 41871

it can be used to map vtk ids to ansys one

vtkId 0 1 2 ansysId 41995 49740 41871
