Detailed Description

vtkALBAPolyDataNavigator.


Useful methods for navigating and editing vtkPolyData and vtkIdList.

Methods already provided by vtkPolyData are:
GetCellPoints(cellid, idlist) to get the points on a cell
GetPointCells(ptid, idlist) to get the cells around a point
GetCellEdgeNeighbours(cellId, id0, id1, idlist) to get the cell neighbours across a cell edge.

Formats of lists:
Edges are listed as EdgeVector = std::vector<Edge>,
and Id's of points and cells as vtkIdList.
However we also use std::multimap and std::set if efficiency demands.

Get the part of idList from indices i0 to i1 inclusive.


The list is assumed to be circular, so n wraps around to 0.

Forwards direction:
If i0 <= i1, you get i0...i1.
If i0 > i1 , you get i0...n-1, 0,1...i1.

Backwards direction:
If i0 >= i1, you get i0...i1, counting backwards.
If i0 < i1, you get i0...0, n-1...i1.
