50//
51// Handle 0 is S (green)
52// " 1 is E (blue)
53// " 2 is W (magenta)
54// " 3 is N (red)
55//
56// The widget should not control the model view's visual pipe components directly.
57// It sets the spline values, which are read by the model view when updating.
58// The model view uses the spline values to calculate the transforms which control the view.
59//
60// Transforms applied to widget and model view components (C is translation by original contour centres)
61// Rot. handle CP
62// Contour axes CPR
63// Cutting planes transform C T^-1
64// Contour segments CPRST C^-1
65// Widget plane CPR
66//
67// Note that because T positions the cutting planes, which affect S,
68// the user cannot return to T after an S has been performed.
69//------------------------------------------------------------------------------
