17/* DESCRIPTION
18vtkALBAContourVolumeMapperGPU - a mapper for direct rendering of isosurfaces &
19a filter for extracting them.
20
21vtkALBAContourVolumeMapperGPU performs a direct rendering of isosurfaces and
22extracts them as polydata objects if requested.
23The input for the class can be either structured points (vtkImageData) or
24rectilinear grids (vtkRectilinearGrids). The optional output is vtkPolyData object.
25This class can produce two kinds of surfaces: a standard isosurface and a
26simplified one that is obtained by skipping some of the voxels.
27It is not guaranteed that the simplified model will have the same topology as the original one.
28This mapper uses a special representation (min-max blocks) of the volume to speed up rendering.
29Additional acceleration is achieved by avoiding recalculation of vertices
30when the same edge is processed again during next iteration.
31This also solves the problem of locating identical points - the Achilles' heel
32of the standard VTK implementation of Marching Cubes algorithm.
33*/
34

36vtkVolumeMapper vtkContourFilter vtkMarchingCubes
