Detailed Description

Class Name: albaOpGarbageCollectMSFDir.

Remove garbage files in MSF directory. These are usually left by do/undo stuff not synchronized with storage. It will also clear the undo stack.

Sample usage:

albaOpGarbageCollectMSFDir *op = new albaOpGarbageCollectMSFDir(); op->SetInput(anyVMETreeNode); int returnValue = op->GarbageCollect();

Removed files absolute paths are written to the log area.
