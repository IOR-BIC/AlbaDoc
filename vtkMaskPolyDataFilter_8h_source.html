12// .NAME vtkMaskPolyDataFilter - Mask vtkDataSet with user-specified polygonal dataset
13// .SECTION Description
14// vtkMaskPolyDataFilter is a filter to mask through data using any subclass of
15// vtkPolyData. That is, scalars are masked where signed distance to polygonal dataset
16// D(x,y,z) <= value, where you can specify a value to mask with.
17// vtkMaskPolyDataFilter computes the distance from the geometry
18// to the points of an input dataset. This distance
19// function can then be "contoured" to generate new, offset surfaces from
20// the original geometry. An important feature of this object is
21// "capping": after the implicit model is created,
22// the values on the boundary of output dataset are set to
23// the maximum distance value. This is used to force closure of the resulting contoured
24// surface. Note, however, that large cap values can generate weird surface
25// normals in those cells adjacent to the boundary of the dataset. Using
26// smaller cap value will reduce this effect.
27
28// .SECTION See Also
29// vtkMaskFilter vtkImpicitModeller
