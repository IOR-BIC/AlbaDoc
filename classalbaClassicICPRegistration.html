Detailed Description

class name: albaClassicICPRegistration Match two surfaces using the iterative closest point (ICP) algorithm.

The core of the algorithm is to match each vertex in one surface with the closest surface point on the other, then apply the transformation that modify one surface to best match the other (in a least square sense). This has to be iterated to get proper convergence of the surfaces.
