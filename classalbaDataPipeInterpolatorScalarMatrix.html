Detailed Description

Class Name: albaScalarMatrixInterpolator.

Data interpolator specialized for albaVMEScalarMatrix (for VNL matrix data). This interpolator is specialized for scalar data. By default selects the right VMEItem, extracts the inner VNL matrix and set it as input of the DataPipe.

See also
