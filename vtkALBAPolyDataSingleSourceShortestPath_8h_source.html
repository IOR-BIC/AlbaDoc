11// .NAME vtkALBAPolyDataSingleSourceShortestPath - Single source shortest path on a polygonal mesh
12// .SECTION Description
13//
14// vtkALBAPolyDataSingleSourceShortestPath is a filter that takes as input
15// a polygonal mesh and performs a single source shortest path
16// calculation. Dijkstra's algorithm is used. The implementation is
17// similar to the one described in Introduction to Algorithms (Second
18// Edition) by Thomas H. Cormen, Charles E. Leiserson, Ronald
19// L. Rivest, and Cliff Stein, published by MIT Press and McGraw-Hill.
20// Some minor enhancement are added though. All vertices are not pushed
21// on the heap at start, instead a front set is maintained. The heap
22// is implemented as a binary heap.
23//
24// The output of the filter is a set of lines describing the shortest path
25// from StartVertex to EndVertex.
26//
27// Thanks to Alexandre Gouaillard and Shoaib Ghias for bug fixes and enhancements
28
