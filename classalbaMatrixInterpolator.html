Detailed Description

Class Name: albaMatrixInterpolator.

albaMatrixInterpolator is used for computing the position of VME in terms of matrix that is an element of vector of items. albaMatrixInterpolator is a matrix pipe producing as output a matrix interpolated against the ones stored in the array associated to the VME itself (only for VMEs with matrix array like albaVMEGenericAbstract).
