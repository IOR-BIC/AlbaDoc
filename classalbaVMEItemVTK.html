Detailed Description

albaVMEItemVTK - store the single VTK dataset stored into a albaDataVector albaVMEItemVTK is an object that stores the single time stamped dataset of a albaVME.

This class associates a Time stamp and a Tagged list to an internally stored dataset. The type of dataset is not defined by base class, and subclasses should define a GetData() and SetData() functions. A albaVMEItemVTK item has also a the following member variables:

m_DataType<\B> to store a "textual description of data". m_Id<\B> to store a numeric unique Id (for internal use) see albaVMERoot::GetNextItemId) m_URL<\B> internally used to store the name of the file where data is stored

See also
albaVMEGenericVTK albaDataVector albaTagArray
