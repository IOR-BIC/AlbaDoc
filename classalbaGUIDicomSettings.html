Detailed Description

Class Name : albaGUIDicomSettings.

Class that contain specific variables regard DICOM, that can be changed from Application Settings, in GUI.

  • Dictionary path
  • DICOM type
  • time bar enable
  • number of slice enable
  • step
  • conversion units
  • rescale to 16Bit
