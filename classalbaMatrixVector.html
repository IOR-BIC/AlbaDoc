◆ SetMatrix() [1/2]

void albaMatrixVector::SetMatrix ( const albaMatrix &  mat)

Set the matrix for a specified time.

If no key-matrix with the same time exist a new key-matrix is created. If a key-matrix with the same time exist, simply overwrite the old 4x4 Matrix value. The matrix content is always copied and NOT referenced.
