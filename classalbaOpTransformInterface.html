Detailed Description

Abstract base class for transform operations.

This class defines the basic structure to assemble components for vme transformations. Typical object that can be plugged in albaOpTransformInterface concrete descendants are gizmos (albaGizmoXXX)and gui transform components (albaGUITransformXXX). See albaOpTransformOld for a concrete implementation.

See also
albaOpTransformOld, albaGUITransformInterface, albaGizmoInterface
