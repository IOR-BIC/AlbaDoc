Detailed Description

Class Name:vtkALBADistanceFilter.

vtkALBADistanceFilter is a filter that computes distance vectors at specified point positions from isosurfaces. The filter has two inputs: the Input and Source. The Input geometric structure is passed through the filter. The vector distances are computed along normals at the Input point positions from isosurfaces of defined value obtained by interpolating into the source data using a search step. For example, we can compute distances on a geometry (specified as Input) from an isosurface in a volume (Source).
