Detailed Description

vtkALBASubdividePolylines.


Filter which subdivides lines into smaller line segments.
Useful if you need to apply several colours to a long line.
The segment size can be set as an absolute value or as the
number of subdivisions.

NB This is meant to be applied to polyline data. If there are cells with more than 2 vertices, their edges will also be subdivided.
