Detailed Description

data interpolator specialized for albaVMEGenericVTK (for VTK data).

This interpolator is specialized for VTK datasets. By default selects the right VMEItem, extracts the inner VTK dataset and set it as input of the vtkALBADataPipe .

See also
vtkALBADataPipe albaVMEGenericVTK
