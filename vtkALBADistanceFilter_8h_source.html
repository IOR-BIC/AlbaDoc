16// .NAME vtkALBADistanceFilter - computes distances from isosurfaces. It can also be used to compute densities.
17// .SECTION Description
18// vtkALBADistanceFilter is a filter that computes distance vectors
19// at specified point positions from isosurfaces. The filter has two inputs:
20// the Input and Source. The Input geometric structure is passed through the
21// filter. The vector distances are computed along normals at the Input point positions
22// from isosurfaces of defined value obtained by interpolating into the source data using a search step.
23// For example, we can compute distances on a geometry
24// (specified as Input) from an isosurface in a volume (Source).
25
