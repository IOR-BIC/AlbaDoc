Detailed Description

class name: vtkBoundingBox.

Fast Simple Class for dealing with 3D bounds. vtkBoundingBox maintains a 3D axis aligned bounding box. It is very lite weight and many of the member functions are in-lined so its very fast It is not derived from vtkObject so it can be allocated on the stack
