Detailed Description

vtkALBAImageFillHolesRemoveIslands: Fill holes or remove island of the specified size (or less) from the given image data vtkALBAImageFillHolesRemoveIslands is a vtkStructuredPointsToStructuredPointsFilte that fill holes or remove islands from the specified vtkStructuredPoint that must be a binary image represented by a vtkUCharArray with values of 0 or 255 only.
