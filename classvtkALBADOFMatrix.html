Detailed Description

class name: vtkALBADOFMatrix vtkALBADOFMatrix holds informations on constrains to be applied on an actor during interaction.

Information for the single dof (ie an element of the internal DOFMatrix[3][3]) is stored in the element struct. The single dof can have different states: LOCK: no interaction is allowed
FREE: free movement BOUNDS: free movement in a given interval SNAP_STEP: snap by a given step SNAP_ARRAY: snap on a given array of positions
