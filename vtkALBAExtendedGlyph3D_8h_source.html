18// .NAME vtkALBAExtendedGlyph3D - copy oriented and scaled glyph geometry to every input point
19// .SECTION Description
20// vtkALBAExtendedGlyph3D is an extended version of vtkGlyph3D supporting visibility by scalar,
21// i.e. the glyph is visualized only if scalar is != 0. Beware, if visibility is ON, scaling and
22// color by scalar are ignored.
23
24// .SECTION See Also
25// vtkTensorGlyph vtkGlyph3D
