3// .NAME vtkALBAImplicitPolyData - treat polygons in input as implicit planes
4// .SECTION Description
5// vtkALBAImplicitPolyData provides the basis for using arbritrary polygonal data
6// as an implicit surface, points are evaluated against nearest polygons which
7// are handled as implicit planes.
8// vtkALBAImplicitPolyData is a concrete implementation of the abstract class
9// vtkImplicitFunction.
10// An internal instance of vtkTriangleFilter is used to filter vertices and
11// lines out of the input PolyData, and a MYvtkPointLocator is used to find the
12// nearest triangle to a candidate point.
