Detailed Description

class name: vtkTriangleQualityRatio This class check the quality of each triangle of a polydata, according to a simple algorithm: qualityLocal = 2.0 * sqrt(3.0)/0.5 * perimeter * longestEdge / area; and assign each value as a scalar to the correspondent triangle.
