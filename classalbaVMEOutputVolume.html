Detailed Description

NULL output for VME node with a VTK image output data.

albaVMEOutputVolume is the output produced by a node generating an output with a VTK dataset.

Internally used to create a new instance of the GUI.

The last id value must be defined as "LAST_ID" to allow the subclass to continue the ID enumeration from it.
