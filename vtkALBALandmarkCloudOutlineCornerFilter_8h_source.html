15// .NAME vtkALBAPointsOutlineCornerFilter - create wireframe outline corners for a Landmark Cloud
16// .SECTION Description
17// This filters differ from vtkOutlineCornerFilter because uses all point of the input source instead of
18// the subset that is contained in cell structure, this is because the landmark cloud manage show
19// of the landmarks by adding/eliminating the cells on the output, but a landmark show/hide should not change
20// the outline of the landmark.
21// This filter also manage the radius of the cloud by enlarging the points bounds of the size of the cloud radius
22// in each dimension.
