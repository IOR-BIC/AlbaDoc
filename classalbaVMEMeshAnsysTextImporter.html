Detailed Description

.NAME albaVMEMeshAnsysTextImporter

Build a albaVMEMesh from information contained in some text files. This text files are actually ANSYS list files without any header, only numbers. Supported elements type are: linear tetra, parabolic tetra, linear hexa, parabolic hexa

BEWARE Hybrid meshes are not upported

See albaVMEMeshAnsysTextImporterTest for use cases
