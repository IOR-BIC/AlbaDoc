16// .NAME vtkALBADOFMatrix
17// .SECTION Description
18//
19// vtkALBADOFMatrix holds informations on constrains to be applied on an
20// actor during interaction. Information for the single dof
21// (ie an element of the internal DOFMatrix[3][3]) is stored in
22// the element struct. The single dof can have different states:
23// LOCK: no interaction is allowed
24// FREE: free movement
25// BOUNDS: free movement in a given interval
26// SNAP_STEP: snap by a given step
27// SNAP_ARRAY: snap on a given array of positions
28//
29// This state is registered in the state variable of element struct.
30//
31// DOFMatrix[3][3]
32// ----------------------------------------------
33// | |
34// | t axis |
35// | r ------------------------- |
36// | a | X Y Z |
37// | n -----------|------------------------ |
38// | s t |TRANSLATE |[0,0] [0,1] [0,2] |
39// | f y |ROTATE |[1,0] [1,1] [1,2] |
40// | o p |SCALE |[2,0] [2,1] [2,2] |
41// | r e |
42// | m |
43// | |
44// | DOFMatrix[i][j] = element |
45// -----------------------------------------------
46//
47// Default values for all elements is LOCK
48//
49// .SECTION See Also
50// albaISAGeneric.h
51//
52// .SECTION ToDo
53//
