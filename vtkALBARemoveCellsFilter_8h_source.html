20// .NAME vtkALBARemoveCellsFilter- actually remove a cell from a vtkPolyData object
21// .SECTION Description
22// vtkALBARemoveCellsFilter removes cells one at a time. vtkCellPicker could be used to get the CellId of the cell to
23// be removed. Then call the RemoveCell method with the CellId. The UndoRemoveCells method restores the
24// PolyData to it original state. Alternatively, you can mark cells with MarkCell and remove them all in one go
25// with RemoveMarkedCells. This is much more efficient for large datasets since the filter only has to execute
26// once.
27
28// .SECTION Caveats
29// Trying to remove the last remaining cell in a dataset will restore all the removed cells!
30// Any filter between this filter and the vtkPolyDataMapper should not alter the cell order
31
32// .SECTION Thanks
33// Goodwin Lawlor, University College Dublin
34
