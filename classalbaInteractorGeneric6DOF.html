Detailed Description

implements 6DOF move of objects in the scene.

This class implements a 6DOF move of objects in the scene. The interaction modality is a direct manipulation, where movement of the tracker are mapped into movements of the object. More details to be written...

Enable/Disable differential moving.

If enable this interactor moves the object by applying a differential transformation to its original pose, i.e. the original pose is retained and only the delta transform is applied to it. Default is false.
