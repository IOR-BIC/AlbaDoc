17// .NAME mflClassicICPRegistration - Implementation of the ICP algorithm.
18// .SECTION Description
19// Match two surfaces using the iterative closest point (ICP) algorithm.
20// The core of the algorithm is to match each vertex in one surface with
21// the closest surface point on the other, then apply the transformation
22// that modify one surface to best match the other (in a least square sense).
23// This has to be iterated to get proper convergence of the surfaces.
24// .SECTION Note
25// Use vtkTransformPolyDataFilter to apply the resulting ICP transform to
26// your data. You might also set it to your actor's user transform.
27// .SECTION Note
28// This class makes use of vtkLandmarkTransform internally to compute the
29// best fit. Use the GetLandmarkTransform member to get a pointer to that
30// transform and set its parameters. You might, for example, constrain the
31// number of degrees of freedom of the solution (i.e. rigid body, similarity,
32// etc.) by checking the vtkLandmarkTransform documentation for its SetMode
33// member.
34// .SECTION see also
35// vtkLandmarkTransform
