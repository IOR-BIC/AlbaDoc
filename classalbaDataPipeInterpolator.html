Detailed Description

superclass for computing the VME output data out of the internal VME-Items.

albaDataPipeInterpolator is a data pipe producing as output a dataset interpolated against the ones stored in the array associated to the VME itself (only for VMEs with data array like albaVMEGeneric). The class also provides the mechanism to obtain the 3D bounds for any time.

See also
mflDataPipe albaVMEGeneric
