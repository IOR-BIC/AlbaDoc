Detailed Description

albaOpMML3GlobalRegistration.


Helper class for albaOpMML3.
This class performs global registration given 0, 3 or 4 pairs of landmarks.
If there are not 3 or 4 pairs, the transform defaults to the identity.
The user sets pairs of landmarks 1 - 3 from atlas and patient, and optionally pair 4.
The class calculates the transform required to move the atlas landmarks to their positions in the patient.
The transform may be applied to polydata or points.
