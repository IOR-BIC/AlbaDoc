Detailed Description

this class create a grid to be used to represent one of the planes z=k, y=k, z=k.

The plane may be specified using SetGridNormal() and k may be specified with SetGridPosition().

To work correctly this actor must be hidden when ComputeVisiblePropBounds is called to Reset the Camera, and must be Shown when ComputeVisiblePropBounds is called to compute ClippingRange. (sorry)
