18// .NAME vtkBox - implicit function for a bounding box
19// .SECTION Description
20// vtkBox computes the implicit function and/or gradient for a axis-aligned
21// bounding box. (The superclasses transform can be used to modify this
22// orientation.) Each side of the box is orthogonal to all other sides
23// meeting along shared edges and all faces are orthogonal to the x-y-z
24// coordinate axes. (If you wish to orient this box differently, recall that
25// the superclass vtkImplicitFunction supports a transformation matrix.)
26// vtkCube is a concrete implementation of vtkImplicitFunction.
27//
28// .SECTION See Also
29// vtkCubeSource vtkImplicitFunction
