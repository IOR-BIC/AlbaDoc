Detailed Description

Simple matrix and vector arithmetic.


This is not itself a vector or matrix; it is only a set of useful methods,
and contains no internal data except a flag for homogeneous mode.

These methods are for vectors and matrices with dimensions 3x3 or 4x4 homogeneous.
Use SetHomogeneous() to set the homogeneous mode (default is off).
NB Many methods assume that the homogenous coord h = 1.

The matrix format is either column-major 1D array format, which is consistent with OpenGL:
0 3 6 or 0 4 8 12
1 4 7 1 5 9 13
2 5 8 2 6 10 14
3 7 11 15

or standard 2D array: A[3][3] or A[4][4].
4x4 matrices are assumed to be homegeneous.

Methods are provided to convert between 1D array and 2D array form.
2D arrays are assumed to be in the familiar row-major form arr2D[row][col], which is consistent with vtkMath.

Matrix convention in this code: i is the row, j is the column, and k is the 1D array index.
The output matrix can be the same as the input unless stated otherwise.

◆ MultiplyMatrixByScalar()

void vtkALBAMatrixVectorMath::MultiplyMatrixByScalar ( double  s,
const double *  A,
double *  B 
) const

Multiply matrix by scalar.


If the matrix is homogeneous, the right column is not multiplied, so that
we preserve the scaling relation S(A)*x = A*S(x),
where x is a vector and S() is the homo scaling operation.
The homogeneous definition does not commute,
so S(A)*x != S(A*x) and A*S(B) != S(A)*B.

Divide matrix by scalar.

The right column is not divided, so that
we preserve the scaling relation S(A)*x = A*S(x),
where x is a vector and S() is the homo scaling operation.
The homogeneous definition does not commute,
so S(A)*x != S(A*x) and A*S(B) != S(A)*B.
